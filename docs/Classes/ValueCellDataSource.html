ValueCellDataSource

open class ValueCellDataSource : NSObject, UICollectionViewDataSource, UITableViewDataSource

A type-safe wrapper around a two-dimensional array of values that can be used to provide a data source for UICollectionViews and UITableViews. There is no direct access to the two-dimensional array, and instead values can be appended via public methods that make sure the value you are add to the data source matches the type of value the table/collection cell can handle.
